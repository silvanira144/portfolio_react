Link do Projeto Publicado no GitHub Pages: https://silvanira144.github.io/portfolio_react/ Descrição Técnica do MVP O portfólio foi desenvolvido utilizando as seguintes tecnologias e práticas: HTML5 Semântico: Estrutura organizada com cabeçalho, seções de “Sobre Mim”, “Meus Projetos”, “Contato” e rodapé. CSS3 Moderno: Estilização completa utilizando Flexbox e Grid Layout, garantindo um design responsivo para dispositivos móveis. JavaScript Dinâmico: Implementação de um array de objetos para gerenciar os projetos, que são renderizados dinamicamente no HTML. Interatividade: Filtros de categoria para os projetos e formulário de contato
